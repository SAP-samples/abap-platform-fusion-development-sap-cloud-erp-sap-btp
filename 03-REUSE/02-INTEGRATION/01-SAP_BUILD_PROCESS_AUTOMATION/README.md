# What is SAP Build Process Automation?

## Overview

**SAP Build Process Automation** is a **solution for the citizen developer** designed to help users adapt, improve, and innovate business processes using **no-code** or **low-code** capabilities.  
It combines **workflow management** and **robotic process automation (RPA)** into a unified tool, empowering both business users and technologists to automate processes efficiently.

With SAP Build Process Automation, organizations can enable **citizen developers** to design and deploy automations while maintaining governance and extensibility for professional developers.

---

## Key Features

### 🧩 Build and Adapt Processes
- Design or adapt processes using an **intuitive graphical interface**.  
- Create **forms-based workflows** easily with **drag-and-drop functionality**.  

### 🧮 Manage Decision Logic
- Define and manage **decision logic** using **tabular, spreadsheet-like decision tables**.  
- Configure business rules for dynamic and adaptive automation flows.

### 🤖 Automate Repetitive Tasks
- Integrate **robotic process automation (RPA)** to handle repetitive, manual tasks.  
- Embed bots directly within process flows for seamless automation.  

### 🧠 Add Intelligence
- Incorporate **machine learning capabilities** to generate intelligent actions and recommendations.  
- Use AI-based components to improve decision-making and process efficiency.

### 🧭 Unified Experience
- Work from a **single launchpad** and **task center** for a unified experience.  
- Monitor, manage, and execute tasks across all automation projects.

### 🔄 Collaboration and Handover
- Allow **citizen developers** to initiate projects and **professional developers** to extend them.  
- Embed advanced actions and workflows into existing projects for scalable automation.

### 📊 Process Visibility
- Gain **real-time transparency** into process performance.  
- Use **process visibility dashboards** to monitor comprehensive workflows and individual process instances.

---

## Prebuilt Content and Resources

SAP Build Process Automation comes with **prebuilt content** to accelerate implementation, including:

- **Bots** for common automation scenarios  
- **Process steps** and templates  
- **Business rules** for configurable logic  
- **Workflow components** for rapid assembly of end-to-end processes  

---

## Benefits

- Empower **business users** to innovate using **no-code** tools.  
- Accelerate automation projects with **prebuilt content** and **AI-driven insights**.  
- Enable **collaboration** between citizen and professional developers.  
- Improve **efficiency, transparency, and agility** across your enterprise processes.
