# What Is SAP Build Work Zone, Standard Edition?

## Overview

**SAP Build Work Zone, standard edition** enables organizations to create a **unified point of access** for SAP (for example, **SAP Cloud ERP (S/4HANA Cloud)**), **custom-built**, and **third-party applications and extensions**, both in the **cloud** and **on-premise** environments.

It provides a centralized workspace experience that improves **user productivity**, **accessibility**, and **collaboration** by integrating business applications, tasks, and content into a single role-based site.

---

## Key Features

### 🧩 Application Integration
- Provide business users with a **central access point** for all their apps and tasks.  
- Support seamless integration across **various UI technologies** (such as SAPUI5, Fiori, and web apps).  
- Enable connection with **third-party applications** for a unified digital workspace.

### 🎨 Intuitive and Engaging User Experience
- Deliver a **role-based, personalized** business site tailored to each user’s needs.  
- Utilize a **flexible content structure** and **configurable design** for intuitive navigation.  
- Create modern and engaging experiences with minimal development effort.

### 🔗 Integrations
- Integrate with core **SAP Business Technology Platform services**, including:  
  - **SAP Task Center / Inbox** for workflow and task management  
  - **SAP Cloud Identity Services** for authentication and access control  
- Enhance productivity with **secure and streamlined** integration flows.

### ⚙️ Extensibility
- Customize the environment through the **extensibility framework**, allowing:  
  - **Shell plugins** for UI enhancements  
  - **Custom branding** to reflect corporate identity  
- Extend existing features or embed new functionality to meet business-specific requirements.

### 🏢 Multitenancy Support
- Use this service in **tenant-aware (multitenant)** applications.  
- Run applications on a **shared compute unit**, enabling multiple consumers (tenants) to operate securely and efficiently.  
- Support scalable and cost-effective deployment models.

---

## Benefits

- Centralize access to **SAP, custom, and third-party applications**.  
- Deliver a **personalized user experience** that enhances productivity.  
- Simplify access management through **SAP Business Technology Platform integration**.  
- Enable **customization and branding** to match business needs.  
- Support **hybrid and multitenant** enterprise environments.
